# **App Name**: Crédé zenagui

## Core Features:

- Add Customer: Allow adding new customer profiles with contact details and other relevant information.
- Record Debt: Enable recording new debts for each customer, including amount, date, and description.
- Record Payment: Allow recording partial or full payments made by customers, updating the balance accordingly.
- Automatic Balance Calculation: Automatically calculate and update the remaining balance after each payment.
- Customer Debt Overview: Provide a comprehensive overview of each customer's debt history, including debts, payments, and balance.
- Debt Reporting: Generate reports summarizing customer debts and payment history

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and professionalism.
- Background color: Light Blue (#E3F2FD), a very desaturated version of the primary, to provide a calm and clean backdrop.
- Accent color: Teal (#009688) to highlight important actions and notifications.
- Body and headline font: 'Inter', a sans-serif font known for its modern, neutral look and readability.
- Use clear and professional icons to represent different actions and categories.
- Implement a clean, intuitive layout with clear sections for adding customers, recording debts, and payments.
- Use subtle transitions and animations to enhance user experience and provide feedback on actions.
